PSU-thesis-latex-template
=========================

The latex template for PSU thesis  

How to use
==========
Full report  

```sh
$pdflatex thesis.tex
```

Each Chapter  
```
$pdflatex <file>.tex
```

File Structures
==============

|File|Description|
|---|---|
|abstract.tex|บทคัดย่อ|
|acknowledgement.tex|กิตติกรรมประกาศ|
|appendix|ภาคผนวก|
|chapter|บทที่|
|thesis.tex|วิทยานิพนธ์ฉบับเต็ม|
|vitae.tex|ประวัติผู้เขียน|
